## 第二層：精煉 (Refining)

### 產品類型

| 產品            | 化學式   | 主要用途             |
|-----------------|----------|----------------------|
| 碳酸鋰 (Li₂CO₃) | Li₂CO₃   | LFP 電池、陶瓷、玻璃 |
| 氫氧化鋰 (LiOH) | LiOH·H₂O | 高鎳 NMC/NCA 電池    |
| 金屬鋰          | Li       | 航空、核能、特殊合金 |
| 氯化鋰          | LiCl     | 空調、熔鹽           |

### 生產路徑

**硬岩路徑**
```
鋰輝石 (SC6) ──▶ 酸烘/熔煉 ──▶ 碳酸鋰/氫氧化鋰
                     │
                     └── 需要化學品（硫酸、石灰）
```

**鹵水路徑**
```
鹵水 ──▶ 蒸發濃縮 ──▶ 純化 ──▶ 碳酸鋰
              │
              └── 週期 12-18 個月
```

## 供需平衡計算

### 計算公式

```python
def compute_supply_demand_balance(year):
    """
    計算供需平衡
    """
    # 供給側
    supply = {
        "mine_production": get_mine_production(year),  # kt Li
        "refined_output": get_refined_output(year),    # kt LCE
        "recycling": get_recycled_li(year),            # kt LCE
    }

    # 需求側
    demand = {
        "ev_battery": compute_ev_li_demand(year),      # kt LCE
        "ess": compute_ess_li_demand(year),            # kt LCE
        "consumer_electronics": get_ce_demand(year),   # kt LCE
        "industrial": get_industrial_demand(year),     # kt LCE
    }

    total_supply = sum(supply.values())
    total_demand = sum(demand.values())

    balance = total_supply - total_demand

    return {
        "supply": total_supply,
        "demand": total_demand,
        "balance": balance,
        "balance_pct": balance / total_demand * 100
    }
```

## 價格傳導機制

### 傳導路徑

```
鋰輝石價格 → 碳酸鋰價格 → 正極材料價格 → 電池成本 → EV/ESS 價格
    │              │              │            │
    └─ 即時 ─┐     └─ 1-3月 ─┐    └─ 1-2月 ─┐  └─ 3-6月
             │              │             │
             ▼              ▼             ▼
         礦業股價      精煉商股價     電池商股價
```

### 價格敏感度

| 環節         | 鋰價 +10% 影響 |
|--------------|----------------|
| 礦業毛利     | +15-25%        |
| 精煉商毛利   | +8-12%         |
| 正極材料成本 | +3-5%          |
| 電池成本     | +1-2%          |
| EV 價格      | +0.3-0.5%      |

### 影響因素

1. **合約期限**: 長單 vs 現貨比例
2. **庫存水平**: 產業鏈庫存緩衝
3. **議價能力**: 上下游博弈
4. **替代性**: 技術路線轉換
