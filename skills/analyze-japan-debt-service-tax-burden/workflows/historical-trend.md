# 歷史趨勢分析工作流

分析 2015-2025 年日本債務利息支出佔稅收比例的完整歷史趨勢。

## 功能概述

此工作流提供：
- 11 年完整財政數據時間序列（FY2015-FY2025）
- Interest/Tax Ratio 歷史走勢分析
- 稅收、利息支出、債務存量趨勢圖
- 隱含平均利率演變
- 風險分級分布統計

## 執行步驟

### Step 1: 執行歷史趨勢分析

**完整分析（2015-2025）**：
```bash
cd skills/analyze-japan-debt-service-tax-burden
python scripts/generate_historical_trend.py --output-dir ../../output
```

**自定義時間範圍**：
```bash
python scripts/generate_historical_trend.py --start-year 2018 --end-year 2025 --output-dir ../../output
```

**指定輸出檔名**：
```bash
python scripts/generate_historical_trend.py --output-file custom_trend.png --output-dir ../../output
```

### Step 2: 查看輸出

輸出檔案：`output/japan_debt_trend_YYYYMMDD.png`

### Step 3: 圖表內容說明

生成的綜合 Dashboard 包含五個區塊：

| 區塊 | 說明 |
|------|------|
| **主趨勢圖** (上方) | Interest/Tax Ratio 完整歷史曲線，帶風險區間背景 |
| **稅收 vs 利息** (中左) | 稅收與利息支出雙柱狀圖對比 |
| **債務存量** (中右) | 政府債務存量趨勢（面積圖） |
| **隱含利率** (下左) | 債務隱含平均利率演變 |
| **風險分布** (下右) | 各風險等級年度數量統計 |

## 關鍵發現（2015-2025）

### 1. 整體趨勢

根據最新分析：

| 指標 | 數值 |
|------|------|
| **比率範圍** | 12.1% ~ 17.4% |
| **平均比率** | 14.3% |
| **FY2015** | 17.4%（最高） |
| **FY2021** | 12.1%（最低） |
| **FY2025** | 15.0%（當前） |

**長期趨勢**：2015 → 2025 整體下降（17.4% → 15.0%）

### 2. 階段性特徵

**階段一：2015-2019（下降期）**
- 比率從 17.4% 下降至 14.7%
- 原因：稅收成長 + 利息支出下降（超低利率環境）

**階段二：2020-2021（疫情衝擊後恢復）**
- 比率降至歷史最低 12.1%（FY2021）
- 原因：疫情後財政擴張，稅收強勁反彈

**階段三：2022-2025（反彈期）**
- 比率從 12.3% 反彈至 15.0%
- 原因：利率正常化，利息支出快速上升

### 3. 風險評估

**所有年度均處於 🟢 GREEN 區**（<25%）
- 表示財政彈性充足
- 尚未進入警戒區域

**但需注意**：
1. 2024-2025 利息支出快速上升（8.5兆 → 10.5兆）
2. 債務存量持續增長（838兆 → 1324兆，增長 58%）
3. 隱含平均利率極低（~0.8%），未來再融資壓力大

## 常見問題

### Q1: 為何 2020-2021 比率大幅下降？

**A**: 兩個主要原因：
1. **利率環境**：BOJ 維持超低利率，存量債務再融資成本極低
2. **疫情後稅收反彈**：2021 稅收達 67 兆（創歷史新高），分母效應

### Q2: 為何 2024-2025 快速反彈？

**A**:
1. **利息支出飆升**：從 8.5 兆跳升至 10.5 兆（+23.5%）
2. **利率正常化**：10Y JGB 從負利率升至 2%+
3. **債務再定價**：存量債務陸續以更高利率再融資

### Q3: 未來會進入黃色警戒區（25%）嗎？

**A**: 需執行壓力測試：
```bash
python scripts/generate_spiral_chart.py --all --output-dir ../../output
```

根據不同利率情境：
- **基準情境（+100bp）**：5-7 年內可能接近 25%
- **壓力情境（+200bp）**：3-5 年內可能達到 25%
- **極端情境（+300bp）**：2-3 年內可能突破 25%

### 數據驗證

腳本會自動檢查數據合理性：
- 稅收範圍：50-100 兆
- 利息支出：5-35 兆
- 債務存量：800-1500 兆

超出範圍會發出警告。
