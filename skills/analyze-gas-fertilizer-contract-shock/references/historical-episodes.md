## 1. 2021-2022 歐洲天然氣危機

### 背景

- 2021 年下半年起，歐洲天然氣（TTF）價格開始飆升
- 主因：俄羅斯供應減少 + 風電出力不足 + 儲氣量低
- 2022 年 8 月 TTF 一度突破 300 EUR/MWh（歷史新高）

### 天然氣 → 化肥傳導

| 時間 | 事件 |
|------|------|
| 2021-09 | CF Industries、Yara 宣布歐洲廠減產 |
| 2021-10 | 多家化肥廠宣布 force majeure |
| 2022-03 | 俄烏衝突後，Urea 價格創歷史新高（$925/ton） |
| 2022-H2 | 天然氣回落，化肥逐步緩解但仍高於歷史均值 |

### 數據觀察

```
TTF 天然氣：
- 2021-09: ~60 EUR/MWh
- 2021-12: ~100 EUR/MWh
- 2022-08: ~300 EUR/MWh (峰值)

Urea:
- 2021-09: ~450 USD/ton
- 2022-01: ~700 USD/ton
- 2022-03: ~925 USD/ton (峰值)

領先期：約 2-4 個月
```

### 本技能驗證結果（預期）

若使用 2021-2022 數據：
- A 段：TTF 多次觸發 z >= 3 的 shock
- B 段：Urea 在 TTF shock 後 2-4 週開始上漲
- C 段：Lead-lag 應顯示 gas 領先 fert 約 14-28 天
- 結論：narrative_supported, confidence: high

## 2. 2022 俄烏衝突

### 背景

- 2022-02-24 俄羅斯入侵烏克蘭
- 俄羅斯是全球天然氣、化肥主要出口國
- 制裁與反制裁導致供應鏈中斷

### 天然氣 → 化肥傳導

這是「直接供應衝擊」而非「成本傳導」：

| 傳導路徑 | 說明 |
|----------|------|
| 路徑 1 | 俄羅斯天然氣出口減少 → 歐洲生產成本上升 → 減產 |
| 路徑 2 | 俄羅斯化肥出口直接減少 → 全球供給缺口 |
| 路徑 3 | 預期心理 → 恐慌性囤積 → 價格飆升 |

### 特殊性

此案例中，天然氣與化肥幾乎「同時」暴漲：
- 2022-02-24 衝突爆發
- 2022-02-28 TTF 單日漲 30%+
- 2022-02-28 Urea 單日漲 15%+

**Lead-lag 分析可能顯示 lag ≈ 0**，因為兩者受同一事件驅動。

### 本技能處理

- 若 lead-lag 顯示同時變動，應標記為「共同驅動」
- 替代解釋：地緣政治事件同時衝擊兩者
- 這不代表「天然氣→化肥」傳導不存在，只是被更強的共同因素掩蓋

## 3. 2018-2019 中國環保限產

### 背景

- 中國實施嚴格環保政策
- 部分煤化工、化肥廠被要求限產
- 中國是全球最大尿素生產國

### 傳導路徑

這是「政策主導」而非「天然氣成本主導」的案例：

- 環保限產 → 中國尿素產量下降
- 國際尿素供給減少 → 價格上漲
- 與天然氣價格無明顯關聯

### 本技能處理

若分析 2018-2019 數據：
- A 段：美國天然氣可能無明顯 shock
- B 段：Urea 仍可能有 spike（但原因是政策）
- C 段：Lead-lag 可能顯示 fert 領先或無相關

**結論**：narrative_weak，替代解釋為「政策驅動」

## 4. 季節性波動

### 天然氣季節性

| 季節 | 特徵 |
|------|------|
| 冬季 | 需求高峰（取暖），價格通常較高 |
| 夏季 | 需求淡季，價格通常較低 |
| 春秋 | 過渡期，價格波動較小 |

### 化肥季節性

| 季節 | 特徵 |
|------|------|
| 春季 | 北半球施肥季，需求高峰 |
| 秋季 | 次高峰（秋播） |
| 冬夏 | 需求淡季 |

### 季節性錯配

天然氣冬季高峰 vs 化肥春季高峰可能產生「虛假」的領先落後關係：

- 1-2 月：天然氣高（冬季取暖）
- 3-4 月：化肥高（春季施肥）
- 若機械地計算 lead-lag，可能誤判為「天然氣領先化肥 2 個月」

### 處理建議

1. 使用 YoY 變化而非絕對水準
2. 關注「異常波動」（z-score）而非「季節性高點」
3. 對照歷史同期，區分「季節性」與「shock」

| 案例 | 年份 | Gas→Fert 成立？ | 特點 |
|------|------|-----------------|------|
| 歐洲能源危機 | 2021-22 | 是 | 經典成本傳導案例 |

## 6. 驗證建議

使用歷史數據驗證本技能時：

### 6.1 預期成功案例

使用 2021-09 至 2022-06 的 TTF + Urea 數據：
- 預期 A 段、B 段、C 段皆通過
- 預期 signal = "narrative_supported"

### 6.2 預期失敗案例

使用 2018-01 至 2019-06 的 Henry Hub + Urea 數據：
- 預期 A 段可能不通過
- 預期 signal = "narrative_weak" 或 "inconclusive"

### 6.3 回歸測試

建立自動化測試，確保：
- 已知成功案例仍輸出 "narrative_supported"
- 已知失敗案例不輸出 "narrative_supported"
