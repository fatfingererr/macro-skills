# 估值指標定義與解讀

## 指標總覽

| 指標 | 英文名 | 公式 | 適用性 |
|------|--------|------|--------|
| 本益比 | Trailing PE | 價格 / 過去 12 個月盈餘 | 最常用，但受短期盈餘波動影響 |
| 預估本益比 | Forward PE | 價格 / 未來 12 個月預估盈餘 | 含未來預期，依賴分析師 |
| CAPE | Shiller PE / CAPE | 價格 / 10 年平均實質盈餘 | 最穩定的長期估值指標 |
| 股價淨值比 | Price-to-Book | 價格 / 每股淨值 | 資產視角，對資本密集型行業敏感 |
| 股價營收比 | Price-to-Sales | 價格 / 每股營收 | 營收視角，較不受盈餘操縱影響 |
| 企業價值倍數 | EV/EBITDA | 企業價值 / 稅息折舊攤銷前盈餘 | 併購視角，考慮負債 |
| 托賓 Q 比率 | Q Ratio | 市值 / 重置成本 | 學術指標，資產重置視角 |
| 巴菲特指標 | Market Cap / GDP | 總市值 / GDP | 總量視角，巴菲特愛用 |

---

## 本益比 (Trailing PE)

### 定義

```
Trailing PE = 當前股價 / 過去 12 個月每股盈餘 (EPS TTM)
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 最直觀、資料取得容易、即時反應 |
| 缺點 | 受短期盈餘波動影響大（景氣低點時分母很小，PE 反而很高） |
| 歷史中位數 | ~15-17（美股長期） |
| 極端高值 | >25 通常被認為偏高 |

### 使用注意

- **周期性行業**：盈餘波動大，PE 可能誤導
- **成長股**：盈餘低但成長快，PE 高不一定高估
- **一次性損益**：重大減值或處分會扭曲 EPS

---

## 預估本益比 (Forward PE)

### 定義

```
Forward PE = 當前股價 / 未來 12 個月預估每股盈餘
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 納入未來預期，對成長股更公平 |
| 缺點 | 依賴分析師預估，預估常過於樂觀 |
| 歷史中位數 | ~14-16（通常比 Trailing PE 低） |

### 使用注意

- **分析師偏誤**：預估通常過於樂觀，尤其在景氣高峰
- **資料來源**：不同來源的預估可能差異很大
- **適用期間**：歷史較短，約 2000 年後才有完整資料

---

## CAPE (Shiller PE / 景氣循環調整本益比)

### 定義

```
CAPE = 實質股價 / 過去 10 年平均實質每股盈餘
```

其中「實質」表示經 CPI 調整。

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 平滑盈餘周期，長期估值最穩定 |
| 缺點 | 反應較慢、可能錯過短期轉折 |
| 歷史中位數 | ~16-17（1871-2020） |
| 極端高值 | >30 歷史上對應重大頂點 |

### 歷史極端事件

| 時期 | CAPE | 後續 |
|------|------|------|
| 1929-09 | ~33 | 大蕭條，跌 89% |
| 1999-12 | ~44 | 科技泡沫，跌 50%+ |
| 2021-12 | ~40 | 待觀察 |

### 使用注意

- **會計準則變化**：GAAP 變化影響盈餘定義
- **產業結構變化**：科技股佔比上升改變盈餘特性
- **利率環境**：低利率可能支持更高估值

---

## 股價淨值比 (Price-to-Book, PB)

### 定義

```
PB = 當前股價 / 每股淨值 (Book Value per Share)
```

其中淨值 = 總資產 - 總負債

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 資產視角，不受盈餘操縱影響 |
| 缺點 | 對輕資產行業（科技）意義有限 |
| 歷史中位數 | ~2.0-2.5（美股） |
| 極端高值 | >4.0 通常偏高 |

### 使用注意

- **產業差異**：銀行 PB ~1，科技 PB 可能 >10
- **無形資產**：品牌、專利等不在帳面上
- **會計差異**：不同會計準則下淨值定義不同

---

## 股價營收比 (Price-to-Sales, PS)

### 定義

```
PS = 當前市值 / 過去 12 個月總營收
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 營收較難操縱、對虧損公司有意義 |
| 缺點 | 不考慮獲利能力，高毛利與低毛利公司不可比 |
| 歷史中位數 | ~1.5-2.0（美股） |
| 極端高值 | >3.0 通常偏高 |

### 使用注意

- **毛利差異**：軟體公司 PS 5 可能合理，零售 PS 5 很高
- **營收品質**：一次性營收會扭曲
- **成長預期**：高成長公司 PS 高有其合理性

---

## 企業價值倍數 (EV/EBITDA)

### 定義

```
EV = 市值 + 總負債 - 現金
EBITDA = 稅息折舊攤銷前盈餘

EV/EBITDA = 企業價值 / EBITDA
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 考慮負債、併購視角、跨槓桿可比 |
| 缺點 | EBITDA 可能美化獲利、忽略資本支出 |
| 歷史中位數 | ~10-12（美股） |
| 極端高值 | >15 通常偏高 |

### 使用注意

- **槓桿公司**：高負債公司 EV 很高
- **資本密集**：折舊攤銷大的行業 EBITDA 失真
- **資料取得**：需要更多財務細節

---

## 托賓 Q 比率 (Q Ratio)

### 定義

```
Q Ratio = 公司市值 / 資產重置成本
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 學術基礎紮實、考慮重置成本 |
| 缺點 | 重置成本難以精確計算、資料來源有限 |
| 理論值 | Q=1 為均衡（市值=重置成本） |
| 歷史中位數 | ~0.7-0.8 |
| 極端高值 | >1.5 歷史上對應重大頂點 |

### FRED 資料

```
Q Ratio ≈ NCBEILQ027S / TNWMVBSNNCB
```

- NCBEILQ027S: 非金融企業股權市值
- TNWMVBSNNCB: 非金融企業淨值

---

## 巴菲特指標 (Market Cap / GDP)

### 定義

```
Buffett Indicator = 股票總市值 / GDP × 100%
```

### 特性

| 面向 | 說明 |
|------|------|
| 優點 | 總量視角、直觀易懂、巴菲特背書 |
| 缺點 | 忽略海外營收、產業結構變化影響大 |
| 歷史中位數 | ~80-100%（美股） |
| 極端高值 | >150% 歷史上對應重大頂點 |

### 歷史極端事件

| 時期 | 市值/GDP | 後續 |
|------|----------|------|
| 1999-12 | ~150% | 科技泡沫 |
| 2021-12 | ~200%+ | 待觀察 |

### 使用注意

- **全球化**：美國企業海外營收占比越來越高
- **上市佔比**：非上市公司沒有納入市值
- **GDP 口徑**：名義 vs 實質、年化 vs 季度

---

## 指標比較總結

### 穩定性排序（高→低）

1. **CAPE** - 10 年平均最穩定
2. **市值/GDP** - 總量視角較穩定
3. **Q Ratio** - 重置成本視角
4. **PB** - 資產視角
5. **PS** - 營收視角
6. **Trailing PE** - 受短期盈餘影響
7. **Forward PE** - 依賴預估

### 歷史長度排序（長→短）

1. **CAPE** - 1871 年起
2. **Q Ratio** - 1950s 起
3. **市值/GDP** - 1950s 起
4. **Trailing PE** - 1980s 起
5. **PB** - 1980s 起
6. **PS** - 1990s 起
7. **Forward PE** - 2000s 起
8. **EV/EBITDA** - 1990s 起

### 建議組合

**基本組合**（最少資料需求）：
- CAPE + 市值/GDP

**標準組合**（平衡覆蓋面與資料品質）：
- CAPE + 市值/GDP + Trailing PE + PB

**完整組合**（最大覆蓋面）：
- CAPE + 市值/GDP + Trailing PE + Forward PE + PB + PS
