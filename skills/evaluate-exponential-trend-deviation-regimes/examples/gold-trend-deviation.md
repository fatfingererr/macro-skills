# 黃金趨勢偏離度分析範例

本文件說明如何使用此工具分析黃金價格的趨勢偏離度與行情體質。

## 黃金專屬特性

### 歷史峰值參考點

黃金有兩個重要的歷史峰值：

| 日期 | 峰值偏離度 | 歷史脈絡 |
|------|-----------|----------|
| 1980-01-31 | ~288% | 1970s 高通膨終局 |
| 2011-08-31 | ~83% | 後金融危機 QE 擔憂 |
| 2025-12-31 | ~100% | 當前（已超越 2011 峰值） |

### 行情體質分類（1970s-like vs 2000s-like）

#### 1970s-like（去信任法幣/通膨風險主導）

**特徵**：
- 實質利率長期為負
- 通膨或通膨預期持續走升
- 地緣政治緊張
- 美元走弱

**黃金行為**：
- 可以維持極端偏離很長時間
- 波動性增加
- 作為「去信任法幣」的避險工具

#### 2000s-like（成長/資產配置驅動）

**特徵**：
- 通膨相對溫和
- 需求和資產配置驅動較強
- 實質利率不一定長期為負

**黃金行為**：
- 偏離度更容易回歸均值
- 作為「資產配置」的一環

## 使用範例

### 基本分析（不含宏觀因子）

```bash
python scripts/trend_deviation.py \
  --symbol GC=F \
  --start 1970-01-01 \
  --compare-peaks "2011-09-06,1980-01-21"
```

### 完整分析（含宏觀因子與行情體質判定）

```bash
python scripts/trend_deviation.py \
  --symbol GC=F \
  --start 1970-01-01 \
  --include-macro \
  --compare-peaks "2011-09-06,1980-01-21"
```

### 快速模式

```bash
python scripts/trend_deviation.py --symbol GC=F --quick
```

## 宏觀因子說明

### 實質利率（Real Rate）

- **數據來源**：FRED DFII10（10 年 TIPS 收益率）
- **權重**：0.30
- **判定規則**：負值 → 支持 1970s-like

### 通膨預期（Inflation Expectation）

- **數據來源**：FRED T5YIE（5 年 Breakeven 通膨率）
- **權重**：0.25
- **判定規則**：上升趨勢 → 支持 1970s-like

### 美元指數（USD Index）

- **數據來源**：FRED DTWEXBGS（貿易加權美元指數）
- **權重**：0.20
- **判定規則**：下降趨勢 → 支持 1970s-like

### 地緣政治風險（Geopolitical Risk）

- **數據來源**：GPR 指數（或新聞關鍵詞計數替代）
- **權重**：0.25
- **判定規則**：上升趨勢 → 支持 1970s-like

## 歷史案例

### 1970s（典型 1970s-like）

| 因子 | 狀態 | 分數 |
|------|------|------|
| 實質利率 | 深度負利率（-4% to -6%） | 1.0 |
| 通膨 | 高且上升（8%-14%） | 1.0 |
| 地緣風險 | 高（石油危機、越戰） | 1.0 |
| 美元 | 大幅走弱 | 1.0 |
| **總分** | | **1.0** |

**結果**：黃金從 $35 漲到 $850，偏離度達 288%。

### 2000s-2011（混合特徵）

| 因子 | 狀態 | 分數 |
|------|------|------|
| 實質利率 | 低但不一定負 | 0.5 |
| 通膨 | 溫和（2%-3%） | 0.3 |
| 地緣風險 | 中等偏高（911後、伊拉克戰爭） | 0.6 |
| 美元 | 先弱後穩 | 0.5 |
| **總分** | | **0.45** |

**結果**：黃金從 $250 漲到 $1900，偏離度約 85%，之後回調。

## 使用建議

### 體質判定的用途

1. **風險評估**：1970s-like 體質下，高偏離度可能持續更久
2. **策略調整**：不同體質下，停損和倉位策略應有所不同
3. **情境規劃**：考慮體質轉變的可能性和觸發條件

### 不應用於

1. **短期交易訊號**：體質是中長期概念
2. **精確擇時**：體質判定是模糊的，不適合精確擇時
3. **單獨決策依據**：應結合其他分析工具使用

### 體質轉變的觸發條件

**從 1970s-like → 2000s-like**：
- 央行大幅升息，實質利率轉正
- 通膨預期被錨定
- 地緣風險緩和

**從 2000s-like → 1970s-like**：
- 通膨失控
- 央行政策失去公信力
- 重大地緣衝突
