<episode name="2008_financial_crisis">
**2008 金融危機**

```
時間軸：
2007-Q4：CASS Shipments YoY 開始下滑
2008-Q1：CASS Shipments YoY 轉負，創 12 個月新低
2008-Q3：CASS Shipments YoY 創週期低點 (-8%)
2008-Q4：CPI YoY 開始快速下滑
2009-Q1：CPI YoY 接近 0%
```

**數據摘要**：

| 指標 | 轉折時間 | 低點時間 | 低點值 |
|------|---------|---------|-------|
| CASS Shipments YoY | 2008-02 | 2009-01 | -10.2% |
| CPI YoY | 2008-09 | 2009-07 | -2.1% |
| 領先月數 | - | - | 約 6 個月 |

**關鍵觀察**：
- CASS 在金融危機前 6-7 個月就開始發出警訊
- 訊號強度：**高**（YoY 大幅轉負 + 創週期新低）
- 領先時間：約 6 個月
- Shipments 和 Expenditures 同時轉負

**當時的宏觀背景**：
- 次貸危機爆發
- 信貸緊縮
- 需求急劇萎縮

</episode>

<episode name="2020_covid_pandemic">
**2020 新冠疫情**

```
時間軸：
2020-03：CASS Shipments YoY 急劇下滑
2020-04：CASS Shipments YoY 低點 (-12%)
2020-05：CPI YoY 低點 (0.1%)
2020-Q3：CASS 快速反彈
2021-Q1：通膨開始上升
```

**數據摘要**：

| 指標 | 轉折時間 | 低點時間 | 低點值 |
|------|---------|---------|-------|
| CASS Shipments YoY | 2020-03 | 2020-04 | -12.3% |
| CPI YoY | 2020-04 | 2020-05 | 0.1% |
| 領先月數 | - | - | 約 1 個月（異常） |

**關鍵觀察**：
- **異常案例**：供給側衝擊導致領先性失真
- 訊號強度：難以判斷（供給 vs 需求混淆）
- CASS 下降是封城導致，非需求自然減弱
- 後續通膨反彈遠超預期

**教訓**：
- 供給側衝擊時，CASS 訊號需謹慎解讀
- 政策干預（財政刺激）可能扭曲領先關係

<current_positioning>

**當前數據的歷史定位**

使用以下框架評估當前訊號：

```python
def historical_positioning(current_cass_yoy: float, historical_data: pd.Series) -> dict:
    """
    計算當前 CASS YoY 在歷史分布中的位置

    Parameters
    ----------
    current_cass_yoy : float
        當前 CASS Shipments YoY
    historical_data : pd.Series
        歷史 YoY 序列

    Returns
    -------
    dict
        百分位數和歷史對照
    """
    percentile = (historical_data < current_cass_yoy).mean() * 100

    # 找類似歷史時期
    similar_periods = historical_data[
        (historical_data > current_cass_yoy - 1) &
        (historical_data < current_cass_yoy + 1)
    ]

    return {
        'current_value': current_cass_yoy,
        'percentile': percentile,
        'similar_count': len(similar_periods),
        'historical_context': get_historical_context(percentile)
    }

def get_historical_context(percentile: float) -> str:
    """根據百分位提供歷史對照"""
    if percentile < 10:
        return "極低，歷史上僅出現在衰退或危機期間（2008、2020）"
    elif percentile < 25:
        return "偏低，通常對應經濟放緩期"
    elif percentile < 75:
        return "中性區間，需觀察趨勢方向"
    elif percentile < 90:
        return "偏高，經濟活動活躍"
    else:
        return "極高，歷史上對應擴張期或復甦期"
```

**解讀建議**：
- 百分位 < 20%：高度警戒，可能預示通膨放緩
- 百分位 20-80%：中性，關注趨勢方向
- 百分位 > 80%：經濟活躍，通膨壓力可能延續

</current_positioning>

<lessons_learned>

**從歷史案例學到的教訓**

1. **供給側衝擊需特別處理**
   - 2020 疫情案例顯示，供給側導致的 CASS 下降不等於需求減弱
   - 建議：結合 PMI 新訂單、ISM 等需求指標驗證

2. **領先時間會變動**
   - 正常週期約 4-6 個月
   - 危機時期可能縮短（因傳導加速）
   - 政策干預可能延長

3. **多指標驗證更可靠**
   - 單一指標可能受特定因素干擾
   - CASS Shipments + Expenditures 一致性更重要

4. **週期轉折比單月波動重要**
   - 關注「創新低」而非「轉負」
   - 持續時間比幅度更有預測力

5. **結構性變化需考慮**
   - 電商、服務業占比變化
   - 傳統指標的代表性可能下降

</lessons_learned>
