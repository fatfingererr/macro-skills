<pattern_observations>

**規律觀察**

從歷史案例中可觀察到以下規律：

**1. 季節性因素**

| 時間 | 特性 | 訊號可靠度 |
|------|------|-----------|
| 農曆新年前（1-2月） | 季節性補庫 | 較低（假陽性多） |
| 春節後（3-4月） | 需求驗證期 | 較高 |
| Q4 年底 | 補庫 + 年度結算 | 中等 |

**2. 宏觀背景影響**

| 背景 | 訊號效果 |
|------|----------|
| 避險情緒上升（地緣政治） | 較可靠 |
| 需求復甦初期 | 可能失效 |
| 政策刺激期 | 可能失效 |
| 衰退擔憂期 | 較可靠 |

**3. 庫存絕對水位**

| 庫存水位 | 訊號可靠度 |
|----------|-----------|
| > 350,000 噸 | 高 |
| 250,000 ~ 350,000 噸 | 中等 |
| < 250,000 噸 | 較低 |

</pattern_observations>

<usage_recommendations>

**使用建議**

基於歷史分析，建議：

1. **不要單獨使用**
   - 命中率 ~60% 僅略優於隨機
   - 需搭配其他指標交叉驗證

2. **考慮宏觀背景**
   - 政策刺激期/需求復甦期：降低訊號權重
   - 避險情緒期/需求轉弱期：提高訊號權重

3. **季節性調整**
   - 農曆新年前的訊號：謹慎對待
   - 節後 2-3 週的訊號：更有參考價值

4. **關注絕對水位**
   - 庫存 > 300,000 噸時的訊號更可靠
   - 庫存 < 200,000 噸時的訊號可能是假陽性

5. **動態監控**
   - 定期更新命中率統計
   - 若近期命中率顯著下降，考慮調整參數或暫停使用

</usage_recommendations>
