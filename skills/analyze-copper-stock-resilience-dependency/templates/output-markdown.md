## 模板使用說明

### 變數替換

報告中的動態數值以 `{variable}` 格式標記，腳本會自動替換：

```python
template_vars = {
    "as_of": "2026-01-20",
    "copper_price": 12700,
    "copper_trend": "up",
    "resilience_score": 78,
    "beta_equity": 0.62,
    # ...
}

report = template.format(**template_vars)
```

### 生成報告

```bash
python scripts/copper_stock_analyzer.py \
    --start 2020-01-01 \
    --end 2026-01-20 \
    --output report.md \
    --format markdown
```
