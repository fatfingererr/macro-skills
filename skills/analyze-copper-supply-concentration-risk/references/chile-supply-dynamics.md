# 智利銅供應結構與動態

## 智利銅礦概況

### 全球地位

| 指標 | 數值 | 全球排名 |
|------|------|----------|
| 礦場產量（2023） | ~5.26 Mt | #1 |
| 銅儲量 | ~190 Mt | #1 |
| 全球份額（產量） | ~26-27% | - |
| 全球份額（儲量） | ~22% | - |

### 核心矛盾

> **儲量第一 ≠ 供應安全**
>
> 智利擁有全球最大銅儲量，但產量增長已停滯甚至下滑。
> 這個矛盾是本 Skill 分析的核心問題。

## 主要礦區

### Escondida（世界最大銅礦）

| 項目 | 數值 |
|------|------|
| 位置 | Atacama 沙漠 |
| 業主 | BHP (57.5%), Rio Tinto (30%), JECO (12.5%) |
| 年產量 | ~1.1 Mt |
| 全球佔比 | ~5% |
| 開採方式 | 露天 + 地下 |

**品位問題**：
- 2010s 品位從 ~1.0% 降至 ~0.6%
- 需要處理更多礦石維持產量

## 品位下滑問題

### 歷史趨勢

| 年代 | 平均品位 | 說明 |
|------|----------|------|
| 1990s | ~1.5% | 高品位時代 |
| 2000s | ~1.0% | 開始下滑 |
| 2010s | ~0.7% | 顯著下滑 |
| 2020s | ~0.6% | 繼續下滑 |

### 影響分析

**「挖兩倍礦才有同樣產出」**

| 品位 | 產出 1t 銅需處理礦石 |
|------|----------------------|
| 1.5% | 67 t |
| 1.0% | 100 t |
| 0.6% | 167 t |

**連鎖效應**：
1. 處理量增加 → 設備投資增加
2. 能源消耗增加 → 成本上升
3. 水資源需求增加 → 沙漠地區壓力
4. 尾礦量增加 → 環境負擔

## 結構性約束

### 水資源限制

智利北部礦區位於 Atacama 沙漠，水資源極度稀缺：

| 解決方案 | 說明 | 成本影響 |
|----------|------|----------|
| 海水淡化 | 從海岸抽水淡化 | +$0.2-0.5/lb |
| 長距離輸水 | 管線可達數百公里 | +$0.1-0.3/lb |
| 水回收 | 提高回收率 | +$0.05-0.1/lb |

### 電力成本

| 因素 | 說明 |
|------|------|
| 能源結構 | 傳統依賴進口化石燃料 |
| 可再生轉型 | 太陽能潛力大但需投資 |
| 礦區用電 | 佔智利總電力消費 ~40% |

### 勞動力與社會

| 因素 | 說明 |
|------|------|
| 勞工談判 | 定期罷工風險 |
| 社區關係 | 原住民土地權爭議 |
| 環境抗議 | 水權、污染問題 |

## 產量趨勢分析

### 歷史產量（1970-2023）

```
1970:  0.7 Mt ▁
1980:  1.1 Mt ▂
1990:  1.6 Mt ▃
2000:  4.6 Mt ▇
2010:  5.4 Mt █
2018:  5.8 Mt █ ← 峰值
2020:  5.7 Mt █
2023:  5.3 Mt ▇ ← 回撤中
```

### 增長階段

| 階段 | 年份 | 特徵 | CAGR |
|------|------|------|------|
| 擴張期 | 1990-2005 | Escondida 等大礦投產 | +7% |
| 高原期 | 2005-2015 | 產量穩定 | +1% |
| 停滯期 | 2015-2023 | 產量持平/下滑 | -0.5% |

### 結構斷點

**2016-2017 年**：產量增長趨勢由正轉負

判定依據：
- Rolling slope 轉負
- 多個大礦區產量見頂
- 投資週期結束

## 替代方案分析

### ore_grade_mode 選項

#### 1. none（不分析品位）
- 只做產量趨勢分析
- 適用於快速評估

#### 2. country_proxy（間接代理）
- 用產量趨勢推斷品位問題
- 邏輯：高銅價 + 產量停滯 → 供應端問題
- 本 Skill 預設使用

#### 3. mine_level（礦場級數據）
- 需要公司年報/財報數據
- 可計算：ore_processed ≈ output / head_grade
- 更精確但數據獲取成本高
