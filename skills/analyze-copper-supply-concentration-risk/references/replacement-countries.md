# 替代供應國分析：秘魯與剛果金

## 替代依賴度框架

當智利產量無法維持增長時，全球銅供應增量主要依賴：

1. **秘魯** - 地理鄰近、礦業基礎完善
2. **剛果金 (DRC)** - 增速最快、品位較高
3. **其他** - 印尼、蒙古、贊比亞等

---

### 風險因素

#### 1. 社會衝突

| 因素 | 說明 | 近期事件 |
|------|------|----------|
| 社區抗議 | 對礦業環境影響的反對 | Las Bambas 道路封鎖 |
| 原住民權益 | 土地權與補償爭議 | 多個項目延期 |
| 罷工 | 勞工待遇談判 | 2023 多次罷工 |

#### 2. 政治不確定性

| 因素 | 說明 |
|------|------|
| 政府更迭頻繁 | 2016-2023 年多次政變/彈劾 |
| 礦業政策搖擺 | 稅制改革提案反覆 |
| 投資環境 | 企業觀望情緒 |

#### 3. 基礎設施

| 因素 | 說明 |
|------|------|
| 物流瓶頸 | 道路運輸風險（社區封路） |
| 電力供應 | 部分礦區依賴自發電 |
| 水資源 | 安第斯高原水權爭議 |

### 執行係數建議

考慮上述風險，建議對秘魯增量潛力使用 **0.7-0.8** 的執行係數。


近 10 年 CAGR：~9%（全球最快）

### Kamoa-Kakula：遊戲改變者

| 指標 | 數值 |
|------|------|
| 品位 | ~4-6%（全球最高之一） |
| 儲量 | 數十年開採壽命 |
| 擴產潛力 | 目標 800 kt+ |
| 問題 | 電力供應、物流 |

### 風險因素

#### 1. 政治風險

| 因素 | 說明 |
|------|------|
| 政治不穩定 | 選舉爭議、武裝衝突 |
| 礦權審查 | 2022-2023 政府重新審查礦權 |
| 國有化風險 | 增加國家持股的政策傾向 |

#### 2. 安全風險

| 因素 | 說明 |
|------|------|
| 武裝團體 | 東部地區衝突 |
| 非法採礦 | 小規模非法開採 |
| 物流安全 | 運輸線路風險 |

#### 3. ESG 爭議

| 因素 | 說明 |
|------|------|
| 人權問題 | 童工、社區遷移 |
| 環境爭議 | 鈷礦開採相關問題 |
| 供應鏈透明度 | 西方買家審查加嚴 |

#### 4. 基礎設施

| 因素 | 說明 |
|------|------|
| 電力供應 | 依賴進口電力或自發電 |
| 物流 | 內陸國，依賴鄰國港口 |
| 加工能力 | 部分出口為礦石/精礦 |

### 執行係數建議

考慮上述風險，建議對 DRC 增量潛力使用 **0.6-0.7** 的執行係數。

## 替代依賴度計算

### 公式

```
replacement_ratio = (expected_increment_peru + expected_increment_drc) / expected_chile_decline
```

### 計算範例

**假設條件（10 年）**：
- 智利產量延續趨勢斜率 -47,000 t/年 → 缺口 470,000 t
- 秘魯 CAGR 4%，執行係數 0.8 → 增量 ~520,000 t × 0.8 = 416,000 t
- DRC CAGR 9%，執行係數 0.7 → 增量 ~870,000 t × 0.7 = 609,000 t

```
replacement_ratio = (416,000 + 609,000) / 470,000 = 2.18
```

這個結果看起來很樂觀，但：

1. **執行風險**：兩國都有顯著的政治/社會風險
2. **風險集中**：依賴兩個高風險國家不比依賴智利更安全
3. **時間錯配**：增量需要新項目投產，有時滯

### 風險調整後的解讀

| Ratio | 表面含義 | 風險調整後含義 |
|-------|----------|----------------|
| > 2.0 | 充足餘裕 | 考慮執行風險後可能只是「剛好」 |
| 1.0-2.0 | 基本填補 | 可能出現供應緊張 |
| < 1.0 | 替代不足 | 供應缺口幾乎確定 |

## 其他替代來源

### 潛在增量國

| 國家 | 2023 產量 | 潛在項目 | 增量潛力 |
|------|-----------|----------|----------|
| 印尼 | ~700 kt | Grasberg 地下 | +100 kt |
| 蒙古 | ~600 kt | Oyu Tolgoi 地下 | +200 kt |
| 贊比亞 | ~800 kt | Kalumbila 擴建 | +100 kt |
| 巴拿馬 | ~350 kt | (已關閉 Cobre Panama) | -350 kt |

### 限制因素

- **印尼**：Grasberg 轉地下後產量穩定但難大幅增加
- **蒙古**：地緣政治風險（夾在中俄之間）
- **贊比亞**：電力問題、投資環境
- **巴拿馬**：Cobre Panama 2023 年底被迫關閉
