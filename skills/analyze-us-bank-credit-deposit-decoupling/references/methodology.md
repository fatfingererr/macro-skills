### 1.1 傳統銀行信貸創造

在傳統銀行體系中，貸款創造存款（Loans Create Deposits）：

```
銀行發放 $100 貸款
    ↓
借款人帳戶增加 $100 存款
    ↓
銀行資產端：貸款 +$100
銀行負債端：存款 +$100
```

**關鍵假設**：在封閉系統內，新增貸款 ≈ 新增存款

### 1.2 脫鉤發生的條件

當以下情況發生時，信貸-存款關係會脫鉤：

| 情況         | 機制                       | 效果               |
|--------------|----------------------------|--------------------|
| **QT 縮表**  | Fed 減少資產負債表規模     | 系統性存款減少     |
| **RRP 吸收** | 貨幣市場基金將資金存入 Fed | 存款流出銀行體系   |
| **TGA 增加** | 財政部帳戶餘額增加         | 存款從銀行流向 Fed |

---

## 2. Decoupling Gap 計算

### 2.1 核心公式

以分析起始日 t₀ 為基準：

```
loan_change(t) = loans(t) - loans(t₀)      # 累積新增貸款
deposit_change(t) = deposits(t) - deposits(t₀)  # 累積新增存款

decoupling_gap(t) = loan_change(t) - deposit_change(t)
```

### 2.2 解讀

| Gap 狀態       | 意義                                       |
|----------------|--------------------------------------------|
| Gap > 0 且擴大 | 貸款創造的存款「消失」了，被某處吸走       |
| Gap ≈ 0        | 傳統信貸創造機制正常運作                   |
| Gap < 0        | 存款增加超過貸款（可能來自 QE 或財政注入） |

---

## 3. Deposit Stress Ratio

### 3.1 定義

```
deposit_stress_ratio = decoupling_gap / loan_change
```

### 3.2 解讀

| Ratio     | 壓力等級 | 意義                         |
|-----------|----------|------------------------------|
| < 0.3     | 低       | 正常信貸創造，存款跟上貸款   |
| 0.3 - 0.5 | 中       | 輕度脫鉤，部分存款流失       |
| 0.5 - 0.7 | 高       | 顯著脫鉤，銀行需搶存款       |
| > 0.7     | 極高     | 嚴重負債端壓力，融資成本上升 |

### 3.3 示例

若在分析期間：
- 累積新增貸款：2.1 兆美元
- 累積新增存款：0.5 兆美元
- Decoupling Gap：1.6 兆美元

則：
```
deposit_stress_ratio = 1.6 / 2.1 = 0.76
```

**解讀**：76% 的新增貸款沒有對應的存款創造，銀行面臨嚴重的負債端壓力。

## 4. RRP 中和效果驗證

### 4.1 假設

若 Gap 主要由 RRP 吸收解釋，則：

```
decoupling_gap ≈ rrp_change
```

### 4.2 驗證方法

計算相關係數：

```python
correlation = np.corrcoef(decoupling_gap, rrp_change)[0, 1]
```

### 4.3 解讀

| 相關係數    | 解讀                                       |
|-------------|--------------------------------------------|
| r > 0.8     | 高度相關，RRP 是主要驅動因素               |
| r = 0.5-0.8 | 中度相關，RRP 部分解釋                     |
| r < 0.5     | 低相關，需考慮其他因素（TGA、QT 直接效果） |

## 5. 隱性緊縮判定邏輯

### 5.1 判定標準

| 條件組合                                          | 判定                                |
|---------------------------------------------------|-------------------------------------|
| 貸款持續上升 + 存款顯著落後 + Gap 與 RRP 高度相關 | **Hidden Balance Sheet Tightening** |
| 貸款下降 + 存款下降                               | Traditional Credit Tightening       |
| 貸款上升 + 存款同步上升                           | Neutral / Expansionary              |

### 5.2 信心水準

| 信心   | 條件                                                    |
|--------|---------------------------------------------------------|
| High   | deposit_stress_ratio > 0.5 且 RRP correlation > 0.8     |
| Medium | deposit_stress_ratio 0.3-0.5 或 RRP correlation 0.5-0.8 |
| Low    | 條件不明確或數據不完整                                  |

## 6. 宏觀意涵

### 6.1 核心洞察

當偵測到隱性緊縮時：
- 緊縮並非來自銀行不放貸（傳統敘事）
- 而是來自負債端（存款）被政策工具抽乾
- 市場需要「額外搶存款」來支撐既有負債結構

### 6.2 市場影響

| 層面         | 影響                               |
|--------------|------------------------------------|
| **銀行**     | NIM（淨利差）壓力，存款成本上升    |
| **信貸市場** | 信貸條件可能收緊（即使貸款量未縮） |
| **利率**     | 短端利率可能上升，曲線可能反轉     |
| **流動性**   | 系統性流動性收緊                   |

---

## 7. 限制與假設

### 7.1 資料限制

- FRED 數據為週頻，可能錯過日內波動
- RRP 為日頻，需對齊至週頻
- 部分數據有修正（revisions）

### 7.2 模型假設

- 假設封閉系統（忽略跨境資金流動）
- 假設 RRP 是主要的存款吸收來源
- 假設銀行資產負債表調整即時反映

### 7.3 不適用情境

- 金融危機期間（非常規政策干擾）
- QE 環境（存款創造超過貸款）
- 重大法規變更期間
