# 印尼鎳供給結構

<overview>
印尼已成為全球鎳供給的主導力量，從 2020 年的 31.5% 市佔
上升至 2024 年的約 60.2%（S&P Global 口徑）。本文件詳述
印尼鎳產業的結構、關鍵園區與政策框架。
</overview>

<market_position>
**印尼市場地位**

| 年份 | 產量 (kt Ni) | 全球份額 | 來源 |
|------|-------------|----------|------|
| 2015 | ~400 | ~15% | USGS |
| 2018 | ~600 | ~22% | USGS |
| 2020 | ~760 | 31.5% | S&P Global |
| 2022 | ~1,600 | ~48% | S&P Global |
| 2024 | ~2,280 | 60.2% | S&P Global |

**成長驅動因素：**
1. 2020 年礦石出口禁令 → 強制在地加工
2. 大規模 NPI (Nickel Pig Iron) 產能投資
3. HPAL 產能擴張（電池級鎳）
4. 中國資本與技術支持
</market_position>

<policy_framework>
**政策框架**

<policy name="RKAB">
**RKAB (Rencana Kerja Anggaran Biaya)**

| 項目 | 說明 |
|------|------|
| 中文 | 年度工作與預算計劃 |
| 性質 | 礦石開採配額制度 |
| 單位 | **Ore wet tonnes**（非 nickel content） |
| 核發 | Ministry of Energy and Mineral Resources (ESDM) |

**近期 RKAB 配額變化（傳聞）：**

| 年份 | 傳聞配額 | 實際開採 | 備註 |
|------|----------|----------|------|
| 2024 | ~400 Mt | TBD | |
| 2025 | ~379 Mt (target) | TBD | |
| 2026 | ~250 Mt (rumored) | N/A | 減少 34% |

**⚠️ 重要提醒：**
- RKAB 配額是 **ore wet tonnes**
- 轉換為 Ni content 需考慮品位 (~1.5%) 和水分 (~30%)
- 250 Mt ore ≈ 2.6 Mt Ni content（大約估計）
</policy>

<policy name="Export_Ban">
**礦石出口禁令**

| 時間 | 政策 |
|------|------|
| 2014 | 首次礦石出口禁令 |
| 2017 | 放寬低品位礦石出口 |
| 2020 | 全面禁止礦石出口 |
| 目前 | 維持禁止，強制在地加工 |

**影響：**
- 強制礦石在國內加工
- 刺激 NPI 產能大規模擴張
- 中國投資湧入
</policy>

<policy name="DMO">
**DMO (Domestic Market Obligation)**

| 項目 | 說明 |
|------|------|
| 目的 | 確保國內供給 |
| 要求 | 部分產品需優先供應國內 |
| 適用 | 正在討論擴大適用範圍 |
</policy>
</policy_framework>

<risk_factors>
**風險因素**

<risk name="policy_uncertainty">
**政策不確定性**

- RKAB 配額年度調整
- 出口規則可能變化
- 環境法規趨嚴
- 稅收政策調整
</risk>

<risk name="infrastructure">
**基礎設施風險**

- 電力供應（煤電為主）
- 物流瓶頸
- 港口容量
</risk>

<risk name="environmental">
**環境風險**

- 尾礦處理
- 海洋排放爭議
- 碳足跡問題（歐盟 CBAM）
</risk>

<risk name="concentration">
**集中度風險**

- 單一國家主導供給
- 政策變動影響全球
- 地緣政治風險
</risk>
</risk_factors>
