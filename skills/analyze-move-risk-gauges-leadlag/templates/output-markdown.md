## 欄位說明與填充規則

### 恐慌判定

```python
if mean_reaction < 1.0:
    spooked_verdict_zh = "偏弱 / 未顯著升溫"
    spooked_verdict_en = "not too spooked"
else:
    spooked_verdict_zh = "明顯上升"
    spooked_verdict_en = "spooked"
```

### 領先判定

```python
if lag_vix > 0 and lag_credit > 0:
    lead_verdict = f"領先 {min(lag_vix, lag_credit)}-{max(lag_vix, lag_credit)} 天"
    lead_interpretation = "可能具領先性"
elif lag_vix < 0 and lag_credit < 0:
    lead_verdict = "落後"
    lead_interpretation = "反而落後於其他指標"
else:
    lead_verdict = "無明確領先/落後"
    lead_interpretation = "與其他指標同步"
```

### 方向一致性解讀

```python
if align_vix > 0.6 and align_credit > 0.6:
    align_interpretation = "符合「MOVE 領先帶動其他風險指標走低」的敘事"
elif align_vix > 0.4 and align_credit > 0.4:
    align_interpretation = "中等程度符合敘事"
else:
    align_interpretation = "敘事支持度較低"
```

### 證據列表

```python
reasons_list = "\n".join([f"- {reason}" for reason in reasons])
```

範例：
```markdown
- MOVE 對 JGB 衝擊的平均反應 (+0.8) 低於歷史中位數 (+4.6)
- MOVE vs VIX 交叉相關峰值出現在 +6 天 (r=0.72)
- MOVE vs Credit 交叉相關峰值出現在 +4 天 (r=0.61)
- MOVE 下行時 VIX 同步下行比例達 58%
```

### 注意事項列表

```python
notes_list = "\n".join([f"- {note}" for note in notes])
```

範例：
```markdown
- MOVE 數據使用 DGS10 實現波動率代理（原始數據抓取失敗）
- CDX IG 使用 ICE BofA IG OAS (BAMLC0A0CM) 作為代理
```
