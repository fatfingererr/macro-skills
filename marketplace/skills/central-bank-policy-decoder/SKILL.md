# 央行政策解讀

專業解讀全球主要央行的政策動向，幫助你掌握貨幣政策脈絡。

## 使用時機

- FOMC 會議後解讀政策聲明與記者會
- 分析央行會議紀要的關鍵資訊
- 解讀央行官員的公開談話
- 評估利率決策對各類資產的影響
- 追蹤央行資產負債表變化（QE/QT）

## 支援的央行

| 央行 | 全稱         | 主要政策工具         |
|------|--------------|----------------------|
| Fed  | 美國聯準會   | 聯邦基金利率、QE/QT  |
| ECB  | 歐洲央行     | 主要再融資利率、PEPP |
| BOJ  | 日本央行     | YCC、ETF 購買        |
| BOE  | 英國央行     | 銀行利率、APF        |
| PBOC | 中國人民銀行 | LPR、MLF、存準率     |
| SNB  | 瑞士央行     | 政策利率、外匯干預   |

## 使用方式

在 Claude Code 中輸入：

```
請解讀以下 FOMC 聲明的關鍵變化：
[貼上 FOMC 聲明全文或重點段落]
```

或

```
請分析 Fed 主席鮑威爾今天記者會的主要訊息
```

## 分析框架

### 1. 政策立場解讀
- **鷹派訊號** - 強調通膨風險、暗示升息
- **鴿派訊號** - 關注經濟下行、暗示寬鬆
- **中性立場** - 數據依賴、保持觀望

### 2. 前瞻性指引分析
- 利率路徑暗示
- 資產負債表政策方向
- 條件式承諾解讀

### 3. 經濟評估變化
- 對經濟成長的看法
- 對就業市場的評估
- 對通膨前景的判斷

### 4. 市場影響評估
- 短期利率預期
- 長期利率走勢
- 風險資產影響
- 匯率影響

## 關鍵術語解讀

| 術語         | 意義                 |
|--------------|----------------------|
| "進一步收緊" | 可能繼續升息         |
| "維持限制性" | 高利率持續較長時間   |
| "數據依賴"   | 根據經濟數據決定政策 |
| "雙向風險"   | 通膨與衰退風險並存   |
| "軟著陸"     | 經濟減速但避免衰退   |

## 範例分析

**FOMC 聲明重點變化**

1. 刪除「可能適宜進一步升息」
   → 訊號：升息週期可能結束

2. 新增「通膨雖有下降但仍偏高」
   → 訊號：降息不會太快來臨

3. 維持「將繼續縮減資產負債表」
   → 訊號：QT 持續進行

**綜合評估**
整體基調偏中性略鴿，暗示政策利率已達峰值，但降息時點仍需觀察更多數據...
