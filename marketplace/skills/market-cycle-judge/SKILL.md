# 市場週期判斷

基於經濟數據與市場指標，判斷當前所處的景氣循環階段，並提供對應的資產配置建議。

## 使用時機

- 評估當前經濟所處的週期階段
- 制定中長期資產配置策略
- 理解不同資產類別在各週期的表現規律
- 識別週期轉折的早期訊號

## 景氣循環四階段

### 1. 復甦期 (Recovery)
**特徵**
- GDP 成長由負轉正
- 失業率開始下降
- 通膨維持低檔
- 央行維持寬鬆政策

**資產表現**
- 股票 ⭐⭐⭐⭐⭐ 強勢
- 債券 ⭐⭐⭐ 中性
- 商品 ⭐⭐⭐ 開始回升
- 現金 ⭐ 弱勢

### 2. 擴張期 (Expansion)
**特徵**
- GDP 穩健成長
- 失業率持續下降
- 通膨開始上升
- 央行逐步收緊

**資產表現**
- 股票 ⭐⭐⭐⭐ 仍具吸引力
- 債券 ⭐⭐ 承壓
- 商品 ⭐⭐⭐⭐ 走強
- 現金 ⭐⭐ 較弱

### 3. 過熱期 (Overheating)
**特徵**
- GDP 成長放緩跡象
- 失業率處於低點
- 通膨明顯上升
- 央行積極升息

**資產表現**
- 股票 ⭐⭐⭐ 波動加大
- 債券 ⭐ 弱勢
- 商品 ⭐⭐⭐⭐⭐ 最強
- 現金 ⭐⭐⭐ 吸引力上升

### 4. 衰退期 (Recession)
**特徵**
- GDP 負成長
- 失業率上升
- 通膨下降
- 央行降息救市

**資產表現**
- 股票 ⭐ 弱勢
- 債券 ⭐⭐⭐⭐⭐ 強勢
- 商品 ⭐⭐ 弱勢
- 現金 ⭐⭐⭐⭐ 防禦價值高

## 使用方式

在 Claude Code 中輸入：

```
請根據以下數據判斷當前美國經濟週期：
- GDP 成長：2.1%
- 失業率：3.7%
- CPI：3.2%
- Fed 利率：5.25-5.50%
- PMI 製造業：46.7
- PMI 服務業：51.4
```

## 週期判斷指標

### 領先指標
- 殖利率曲線（倒掛預示衰退）
- PMI 製造業
- 新屋開工
- 消費者信心

### 同步指標
- 非農就業
- 工業生產
- 零售銷售
- 個人所得

### 落後指標
- 失業率
- CPI
- 企業獲利
- 貸款違約率

## 經典投資時鐘

```
        通膨上升
           ⬆
    過熱期 │ 衰退初期
   (商品)  │  (現金)
 ←─────────┼─────────→ 成長放緩
   擴張期  │  復甦期
   (股票)  │  (債券→股票)
           ⬇
        通膨下降
```

## 範例分析

**輸入數據**
- GDP：2.1%（前值 2.4%）
- 失業率：3.7%
- CPI：3.2%
- Fed 利率：5.25-5.50%

**週期判斷**
當前處於「擴張期後期至過熱期轉折」階段

**依據**
1. GDP 仍正成長但動能放緩
2. 失業率處於歷史低檔
3. 通膨雖下降但仍高於目標
4. 利率處於限制性水平

**配置建議**
- 降低股票權重，增加防禦型配置
- 短債優於長債
- 適度持有商品對沖通膨
- 提高現金部位應對波動
