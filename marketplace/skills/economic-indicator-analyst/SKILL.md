# 經濟指標分析師

專業的經濟指標分析助手，幫助你快速理解各類經濟數據的意義與市場影響。

## 使用時機

- 當新的經濟數據公佈時，需要快速解讀其意義
- 分析多個經濟指標之間的關聯性
- 評估經濟數據對金融市場的潛在影響
- 撰寫經濟分析報告時需要專業解讀

## 支援的經濟指標

### 產出與成長
- **GDP（國內生產毛額）** - 季度 GDP 成長率、年化成長率
- **工業生產指數** - 製造業產出變化
- **零售銷售** - 消費者支出趨勢

### 通膨與物價
- **CPI（消費者物價指數）** - 整體通膨與核心通膨
- **PPI（生產者物價指數）** - 生產成本變化
- **PCE（個人消費支出）** - Fed 偏好的通膨指標

### 就業市場
- **非農就業人數** - 月度就業變化
- **失業率** - 勞動市場健康度
- **平均時薪** - 薪資通膨壓力
- **初領失業金人數** - 就業市場即時狀況

### 領先指標
- **PMI（採購經理人指數）** - 製造業與服務業景氣
- **消費者信心指數** - 消費者情緒
- **ISM 製造業指數** - 美國製造業狀況

## 使用方式

在 Claude Code 中輸入：

```
請分析最新公佈的美國 CPI 數據：
- 整體 CPI 年增 3.2%
- 核心 CPI 年增 4.0%
- 月增 0.4%
```

## 分析框架

1. **數據解讀** - 與預期值、前值的比較
2. **趨勢分析** - 短期與長期趨勢判斷
3. **結構分析** - 主要驅動因素拆解
4. **市場影響** - 對股市、債市、匯市的影響
5. **政策啟示** - 對央行政策的潛在影響

## 範例輸出

**數據摘要**
- 整體 CPI：3.2%（預期 3.1%，前值 3.0%）
- 核心 CPI：4.0%（預期 3.9%，前值 4.0%）

**解讀**
通膨數據略高於預期，顯示物價壓力仍具黏性。核心通膨持平表明...

**市場影響**
- 股市：短期承壓，升息預期上升
- 債市：殖利率可能上揚
- 匯市：美元指數可能走強
